# Игра «Найди пару»

## Описание
Игра «Найди пару» – это классическая логическая головоломка, направленная на развитие зрительной памяти и внимательности пользователя. 
Место действия игры – игровое поле, заполненное набором закрытых карточек. 
Каждая карточка скрывает под собой определенное изображение. Игрок может одновременно открыть не более двух карточек. 
Цель игры заключается в том, чтобы за минимальное количество ходов и времени найти и открыть все идентичные пары изображений на игровом поле.

## Установка
1. Перейдите в папку с программой.
2. Убедитесь, что папка `Images` находится в той же директории, что и исполняемый файл.
3. Дважды щёлкните левой кнопкой мыши по файлу `Найди пару.exe`.
4. На экране появится главное окно программы.

## Использование
Игра служит эффективным тренажером для когнитивного развития. В процессе прохождения уровней различной сложности пользователь стимулирует нейронные связи, тренируя оперативную память и способность к концентрации.
Программа находит применение в следующих областях:
- **Образование**: как интерактивный модуль для развития логического мышления у детей.
- **Саморазвитие**: для ежедневных тренировок памяти и внимания.
- **Реабилитация**: в качестве вспомогательного средства для поддержания когнитивных функций.
- **Программирование**: проект демонстрирует принципы событийно-ориентированного программирования на языке C# и работу с графическими ресурсами в Windows Forms.
